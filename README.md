## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

```bash
git clone https://github.com/sonuthakur03/AssemblyAvengers.git
cd AssemblyAvengers
npm install
```

### Run the Project

```bash
npm run dev
```
